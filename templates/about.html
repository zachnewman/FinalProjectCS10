About this Website


We developed this game and website as the final project for our Intro to Python course. We first coded the game as a Python script, then moved it to a website using a webapp framework called Flask, where we coded pages in HTML, CSS, and Jinja. We have extensively tested our game for bugs and have added new features over time. The first build of our website required the player to type in which choice of card they would like to select from a text list. Now, it displays pictures of each card and buttons for selection. We hope you enjoy our game.
